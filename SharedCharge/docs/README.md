## 项目简介

本项目是一个微信小程序，用于为用户推荐最优的充电宝借还方案。通过智能分析用户的起点和终点位置，结合多种排序策略，为用户提供最合适的充电宝品牌选择。

## 功能特点

- 地图交互：用户可通过地图选择起点和终点
- 多维度排序：支持按距离、时间、价格、口碑等多种方式排序
- 智能推荐：根据综合评分推荐最优借还路径
- 导航功能：提供到借充点的导航服务
- 详情展示：美观直观地展示品牌详情
